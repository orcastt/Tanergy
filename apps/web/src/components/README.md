# Components

- `app-shell/`: shared product shell navigation and account entry.
- `workspaces/`: Workspace Board gallery, Board management panel, members/share UI and local/remote Board actions.
- `konva-canvas/`: primary production canvas runtime for `/boards/[boardId]` and public share viewing.
- `canvas/`: gated tldraw reference shell; do not add new production-only behavior here.
- `nodes/`: reusable node card pieces used by current and future canvas runtimes.
- `collection/`: Collection surface components.

Empty or historical component directories should not be treated as active architecture without updating this map.
